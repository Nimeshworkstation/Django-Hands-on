This is one method to get value in session using key keyword in template file

{% for key in keys %} {{key}} {% endfor %}
this is second method by using items keyword that shows all items passed from view function

{% for item in items %} {{item}} {% endfor %}
This is 3rd method showing key and value seprately using item keyword passed from view function

{% for key, value in items %} {{key}}{{value}} {% endfor %}
